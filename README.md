# ATSFácil

SaaS em Next.js 14 para criar currículo ATS em etapas, autenticar com Google via Supabase, cobrar Pix via Mercado Pago e gerar PDF localmente com jsPDF.

## Desenvolvimento

```bash
npm install
npm run dev
```

Variáveis usadas:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `MERCADOPAGO_ACCESS_TOKEN`
- `NEXT_PUBLIC_APP_URL`
